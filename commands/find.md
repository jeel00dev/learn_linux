# find Command

The `find` command is used to **search for files and directories** in a filesystem hierarchy based on name, type, size, time, permissions, and more.

It is one of the most powerful and commonly used Linux commands.

---

## Basic Usage

```bash
find <path> <options>

find by name
find . -name (file name)
find . -iname (file name) # case insensitive

```
