# Tree Command

## Basic Usage

Displays the directory structure of the current directory.

```bash
tree
```

## Options

### Limit Depth

```bash
tree -L 2
```

### Show Only Directories

```bash
tree -d
```

### Include Hidden Files

```bash
tree -a
```

### Ignore Specific File or Directory

```bash
tree -I ".git"
```

### Show File Size

```bash
tree -h
```
