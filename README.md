# learn_linux

A repository for learning intermediate to advanced Linux concepts, with a focus on Linux commands, permissions, and filesystem behavior.

> This project is a work in progress and is primarily maintained as a personal learning reference.  
> Over time, it will be structured and formatted to be more beginner friendly for other learners.

---

## Directory Structure

```text
.
├── LICENSE
├── README.md
├── commands/
│   ├── chmod.md
│   ├── common_cmd.md
│   └── groups.md
    .
    .
    .
└── note.md
```

### Folder & File Explanation

#### commands/

Contains detailed documentation for important Linux commands and related concepts.

Each file focuses on:

- Command purpose
- Syntax
- Permission behavior
- Practical and real life examples

Examples:

- chmod.md → Permissions, special bits, real scenarios
- groups.md → Users, groups, ownership, collaboration
- common_cmd.md → Frequently used Linux commands

#### note.md

Contains small Linux commands and notes that:

- Do not require a dedicated file
